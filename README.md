
# What does it do?

**⚠️ Projet en phase initiale - First Skyrim Mod**

Ceci est mon premier mod Skyrim. L'objectif ultime de ce projet est de créer une **Intelligence Artificielle capable de contrôler entièrement le joueur**.

Actuellement, le projet en est qu'au début du développement. Les fondations sont en place avec un plugin SKSE C++ fonctionnel, mais les principales fonctionnalités d'IA sont encore à implémenter.

### Vision du projet :
- Développer une IA autonome pour diriger les actions du joueur
- Créer une interaction intelligente avec le monde de Skyrim
- Automatiser les décisions et mouvements du personnage

**Status**: 🔨 En construction - Des mises à jour régulières viendront enrichir le mod.
